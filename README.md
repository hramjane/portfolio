# 🚀 Space Collector - GitHub Pages Game

An interactive space-themed game built entirely with vanilla JavaScript and Canvas, hosted on GitHub Pages!

## 🎮 How to Play

- **Move:** Use Arrow Keys or WASD
- **Pause:** Press Space
- **Goal:** Collect golden stars ⭐ to earn points
- **Watch Out:** Avoid gray asteroids ⚫ - you have 3 lives!
- **Progress:** Each level adds more stars and faster asteroids

## 🌟 Features

- Smooth, responsive gameplay
- Progressive difficulty (increases with each level)
- Particle effects for collection and collisions
- Score tracking and level progression
- Beautiful space-themed UI with glow effects
- Fully responsive design (works on desktop and mobile)
- Zero dependencies - pure vanilla JavaScript!

## 📊 Game Mechanics

### Scoring System

- Collect stars for points (10 × current level)
- Higher levels = bigger rewards!

### Difficulty Progression

- Level 1: 3 asteroids, 3 stars
- Each level: More stars, faster asteroids
- Asteroids increase by 0.5× speed per level

### Game Over

- You start with 3 lives ❤️
- Hitting an asteroid costs 1 life
- Game ends when lives reach 0
- Final score and level are displayed

## 🔧 Customization

You can easily modify:

- Canvas size: Edit `resizeCanvas()` in `game.js`
- Colors: Modify `:root` variables in `styles.css`
- Game speed: Adjust `player.speed` and asteroid spawning
- Lives: Change initial `this.lives` value
- Point values: Edit the score calculation
