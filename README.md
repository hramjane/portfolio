# GitHub Pages Demo Portfolio

This is a sample portfolio website hosted on GitHub Pages. It demonstrates how to create and host a static website completely free on GitHub!

## Features

- Responsive design that works on desktop, tablet, and mobile
- Modern HTML5 and CSS3
- Smooth scrolling navigation
- Portfolio section to showcase projects
- Contact section with social links

## Files Included

- `index.html` - Main HTML file
- `styles.css` - Styling
- `script.js` - JavaScript functionality
- `README.md` - This file

## How to Use

1. Fork or clone this repository
2. Customize the content in `index.html` with your information
3. Update links and social media URLs
4. Push to GitHub
5. Enable GitHub Pages in repository settings

## GitHub Pages Setup

Follow these steps to publish your website:

1. Go to your GitHub repository
2. Click **Settings** (top right)
3. Scroll to **Pages** section (left sidebar)
4. Under **Source**, select the branch (usually `main`) and folder (`/root` or `/docs`)
5. Click **Save**
6. Your site will be published at: `https://[username].github.io/[repository-name]/`

For more info, visit: https://docs.github.com/en/pages/getting-started-with-github-pages/about-github-pages

## Customization Tips

- Edit HTML content in `index.html`
- Change colors in `styles.css` (look for `:root` CSS variables)
- Add more projects in the projects section
- Add your social media links in the contact section
